#### 翻译系统

*MITE Resource Pack 1.6.4.zip*中的*MITE.lang*提供的翻译的优先级总是最高的, 因此无论如何切换语言, 游戏总是读取这里的翻译.

本模组做的事情: 将*MITE.lang*的优先级移至最低.

可能存在的问题:

Q: 即使我切换成美式英语, 仍有部分中文?

A: 用英式英语.
