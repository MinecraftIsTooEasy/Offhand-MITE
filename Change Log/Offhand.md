# MITE-Offhand Mod

## 2.0.0

### 新增功能：

    - 完善副手物品使用系统
    - 支持副手使用食物、药水、水桶等物品
    - 副手物品使用时正确播放音效（每4tick播放一次，与原版节奏一致）
    - 改进副手物品渲染系统，支持更多物品类型

### 修复问题：

    - 修复副手使用物品时音效不播放的问题
    - 修复副手使用物品时客户端和服务器端状态不同步的问题
    - 改进副手物品的NBT数据保存和加载
    - 优化副手物品使用的验证逻辑

### 使用说明：

    按F键进行副手交互

    交互规则：
     |------------------------------------|
     |   主手	 副手	  右键操作         |
     |   空	   任何物品	使用副手物品        |
     |   铲子	沙砾	    放置沙砾（不会格挡） |
     |   镐子	圆石	    放置圆石（不会格挡） |
     |   方块	方块	    使用主手方块        |
     |------------------------------------|
